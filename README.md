# Simple Bird Shooting Game

A simple single-player shooting game written in C#.  
The goal is straightforward: **shoot the bird that appears on the screen using your mouse**.  
The player has exactly **10 bullets**, and each shot reduces the ammo count by one. Aim carefully and shoot the bird to score points!
